# SEO Meta Checker

A lightweight SEO auditing tool that analyzes Title Tags and Meta Descriptions across multiple URLs.

## Features

- Extract page titles
- Extract meta descriptions
- Check title length
- Check description length
- Detect missing SEO elements
- Export results to CSV

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Add URLs to:

```text
sample_urls.txt
```

Run:

```bash
python seo_checker.py
```

Results will be saved as:

```text
seo_report.csv
```

## Example Output

| URL | Title Length | Description Length |
|------|------|------|
| example.com | 45 | 155 |
